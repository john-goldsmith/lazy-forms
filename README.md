# Lazy Forms
A jQuery plugin to visually customize native HTML form controls.

## Compiling from source
- Clone this repo
- Run `npm install`
- Run `gulp`

## Dependencies
- jQuery >= 1.4

## Usage
`$('input:checkbox').lazyForms();`
