Lazy Forms

A jQuery plugin to visually customize native HTML form controls

Checkboxes

  • Output
  • Code

HTML

<form action="" method="post">
  <input name="my-checkboxes[]" data-label="Checkbox 1" type="checkbox" value="cb-123" />
  <input name="my-checkboxes[]" data-label="Checkbox 2" type="checkbox" value="cb-abc" />
  <input name="my-checkboxes[]" data-label="Checkbox 3" type="checkbox" value="cb-true" />
</form>

JavaScript

$("input:checkbox").lazyForms();
